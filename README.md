# Bundestag Wrapped

![Bundestag Wrapped - A Year in Review](social-preview.png)

A Spotify Wrapped-style visualization of German Bundestag speeches. Built as a React web app and native mobile app.

## Quick Start

```bash
# Web
npm install && npm run dev

# Mobile
cd mobile && npm install && npm start
```

## Project Structure

```
├── src/                    # Web app (React + Vite)
├── mobile/                 # Mobile app (Expo + React Native)
├── public/                 # Static JSON data
└── Dockerfile              # Production deployment
```

## Tech Stack

| Web | Mobile |
|-----|--------|
| React 19, Vite 7 | Expo SDK 54, React Native 0.81 |
| TailwindCSS 4 | NativeWind |
| Motion | Moti + Reanimated |
| React Router 7 | Expo Router 6 |

## Data

Static JSON in `public/`, generated by [bundestag-analysis](https://github.com/movm/bundestag-analysis):

| File | Description |
|------|-------------|
| `wrapped.json` | Party stats, tone analysis, quizzes |
| `speakers/*.json` | 700+ individual speaker profiles |
| `speeches_db.json` | Full searchable speech database |

## Deployment

```bash
# Docker
docker build -t bundestag-wrapped . && docker run -p 3000:3000 bundestag-wrapped

# Static hosting
npm run build  # Deploy dist/

# Mobile (EAS)
cd mobile && eas build --platform android --profile production
```
